## Ciclo de vida de datos

En esta primera etapa se debe comprender el ciclo de vida de los datos, de dónde provienen, cómo se almacenan y todo el uso que se les puede dar.

Conocer y entender la función de los diferentes roles de data y cómo interactúan entre ellos.

- Data Engineer
- Data Architect
- Data Analyst
- Data Science
- BI Engineer
- ML Engineer
- AI Engineer

- GCP: Bigquery, Cloud Storage, Cloud Function, Airflow
- AWS: Snowflake, S3, Airflow, dbt
- Azure: Databricks, Data Factory
- Entre otros...
